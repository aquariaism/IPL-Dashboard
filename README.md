#IPL Performance Dashboard (2018–2023)

##Project Overview

This project presents an **interactive IPL Data Analytics Dashboard** built using **Microsoft Excel**.
It provides insights into player performance, team contributions, and match-level statistics from IPL seasons **2018 to 2023**.

The dashboard is designed with a **clean UI, interactive slicers, and KPI cards**, making it easy to explore and analyze IPL data dynamically.


## Features

***Top 10 Run Scorers** (Bar Chart)
***Top IPL Teams by Total Runs** (Pie Chart)
***Key Performance Indicators (KPIs):**

  * Total Runs
  * Total Matches
  * Average Runs per Match
  * Highest Run Scorer
* Interactive Filters (Slicers):**

  * Team (Bat First)
  * Venue
  Professional Dashboard Design**

  * Clean layout (no gridlines)
  * Consistent color theme
  * Interactive and user-friendly


## Insights Generated

* Identify top-performing batsmen across seasons
* Compare team-wise total runs
* Analyze performance based on venue and team selection
* Understand scoring trends through average runs per match

## How to Use

1. Download `dashboard.xlsx`
2. Open in Microsoft Excel
3. Use slicers (right side) to filter data:

   * Select teams
   * Choose venues
4. Explore dynamic updates in charts and KPIs
